# Developer Toolkit App

## Overview

The Developer Toolkit App is an Android application built using Kotlin and Jetpack Compose. It is designed to help developers store useful information, including notes and development resources, while also providing basic device information.

This application demonstrates core Android development concepts such as UI design with Jetpack Compose, state management, and navigation.

---

## Features

### Device Information

Displays essential device details:

* Android version
* SDK level
* Device model

---

### Developer Notes

Allows users to:

* Add short notes
* View saved notes
* Delete notes

Notes are stored temporarily using in-memory state.

---

### Developer Links Manager

Users can:

* Save useful links
* Assign a title, URL, and category
* View saved links
* Delete links

#### Categories include:

* Documentation
* Tutorials
* Tools

---

### Navigation

* Implemented using Jetpack Compose Navigation
* Users can switch between screens easily using navigation buttons

---

## Technologies Used

* Kotlin
* Jetpack Compose
* Android Studio
* Navigation Compose

---

## How to Run the App

1. Open the project in Android Studio
2. Sync Gradle files
3. Run the app on an emulator or Android device
